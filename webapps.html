Web apps

Here you can access a collection of web apps, made by me.

Lingle

A Dutch version of Wordle with a familiar look and feel. Supports the Ä² digraph!

Go to Lingle

Tophat

Tophat is a powerful bitmap font creation tool for Playdate game development using the Playdate SDK. It can import TTF/OTF fonts, allows for wide Unicode support, and lets you preview, filter and edit kerning pairs directly from the main screen.

Go to Tophat

Oven

Oven lets you convert images to 1-bit, and is primarily made to help with creating assets for Playdate games. It supports various types of ordered dithering (including blue-noise dithering), and also features a simple brightness curve editor.

Go to Oven

Counter

This thing lets you keep count of anything, saving it to your browser’s local storage. You can add multiple counters, change their names and colours, and all of it will still be there even if you refresh the page.

Go to Counter

Bandgen

Bandgen is a sloppy made‑in‑a‑day does‑the‑job kind of tool that generates 3DS Virtual Console styled banners, made to be used with New Super Ultimate Injector (by Asdolo on GBAtemp) because Wine wouldn't let me use the font Rodin properly.

Go to Bandgen
